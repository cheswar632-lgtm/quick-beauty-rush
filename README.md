# Slump — Flash Deal Salon Marketplace

A flash-deal beauty salon marketplace for Pune, India. Built for SuperXgen AI Startup Buildathon 2026.

## Live Site
[View Live →](https://quick-beauty-rush.lovable.app/)

## The Problem
Luxury salons operate at 90%+ capacity on weekends but under 25% on weekday afternoons — while rent and staff costs run 100% every day.

## The Solution
Slump lets salon owners post "dead hour" flash deals (35-50% off) valid for the next 2 hours. Customers get premium salon services at discounted rates. Both sides win.

## Features
- Live feed of flash deals with countdown timers
- Filter by service: Hair / Skin / Nails / Bridal
- 3-step booking flow (no payment required)
- Salon owner dashboard to post new deals
- Mobile-first responsive design
